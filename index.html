JavaScript Reference Guide

1. Introduction to JS

JavaScript is a High-level, Dynamically-typed, and Interpreted programming language.

High Level

Easy for humans to understand and write. Memory management is handled automatically. Example: Using simple keywords like if (x > 10) { ... } else { ... }.

Dynamically-typed

Smart enough to understand data types automatically without explicitly declaring them. Types can change at runtime. Example:

let num = 5; // Automatically treated as a Number (Not a String)

Interpreted

The code is executed line-by-line (synchronously) by the browser's JS engine, rather than requiring a separate compilation step beforehand.

2. Variables

Variables are containers for storing data values. JS uses three main keywords to declare them:

1. var

The older way to declare variables. It is function-scoped and can be re-declared and updated.

var x = 10; var x = 20; // Allowed: Re-declared without errors

2. let

The modern way to declare variables. It is block-scoped (lives only inside {}) and can be updated, but cannot be re-declared in the same scope.

let score = 90; score = 95; // Allowed: Updated value // let score = 100; // Error: Cannot re-declare

3. const

Short for constant. It is block-scoped and cannot be updated or re-declared. It must be initialized when declared.

const PI = 3.14159; // PI = 3; // Error: Assignment to constant variable

3. Data Types

Data types specify the classification of data that a variable can hold in JavaScript. They are broadly divided into Primitive and Non-Primitive types.

1. String

Represents textual data. Must be wrapped in single or double quotes.

let name = "Noor"; let city = 'Delhi';

2. Number

Represents both integer and floating-point (decimal) numbers.

let age = 25; let price = 99.99;

3. Boolean

Represents a logical entity and can have only two values: true or false.

let isJsFun = true; let isGraduated = false;

4. Undefined

A variable that has been declared but has not yet been assigned a value.

let userData; console.log(userData); // Outputs: undefined

5. Null

Represents the intentional absence of any object value.

let emptyValue = null;

6. Object (Non-Primitive)

Used to store collections of data and more complex entities as key-value pairs.

let student = { firstName: "Noor", age: 22, isPassed: true };

4. Operators

1. Arithmetic Operators

Used to perform mathematical calculations.

let sum = 10 + 5; // Addition (+) -> 15 let diff = 10 - 5; // Subtraction (-) -> 5 let product = 10 * 5; // Multiplication (*) -> 50 let quotient = 10 / 5; // Division (/) -> 2 let remainder = 10 % 3; // Modulus (%) -> 1

2. Assignment Operators

Used to assign values to variables.

let x = 10; // Simple assignment (=) x += 5; // Equivalent to: x = x + 5 (Result: 15) x -= 3; // Equivalent to: x = x - 3 (Result: 12)

3. Comparison Operators

Used to compare two values and return a boolean (true/false).

5 == "5"; // True (Compares values only) 5 === "5"; // False (Strict: Compares both value and data type) 10 > 5; // True (Greater than) 10 != 5; // True (Not equal to)

4. Logical Operators

Used to determine the logic between variables or values.

// Logical AND (&&): Both conditions must be true -> (5 > 3 && 10 > 20) returns false // Logical OR (||): At least one condition must be true -> (5 > 3 || 10 > 20) returns true // Logical NOT (!): Reverses the boolean result -> !(5 > 3) returns false

5. Ternary (Conditional) Operator

Acts as a shortcut for an if...else statement.

Syntax: condition ? value_if_true : value_if_false;

let userAge = 18; let status = (userAge >= 18) ? "Adult" : "Minor"; console.log(status); // Outputs: "Adult"

5. Conditional Statements

Used to execute different actions based on different conditions.

1. if Statement

Executes a block of code if a specified condition is true.

let age = 20; if (age >= 18) { console.log("You can vote."); }

2. if...else Statement

Executes a block of code if the condition is true, and another block if it is false.

let isRaining = false; if (isRaining) { console.log("Take an umbrella."); } else { console.log("Enjoy the sunny day!"); }

3. Else-if Ladder

Used to specify a new condition if the first condition is false.

let marks = 75; if (marks >= 90) { console.log("Grade: A+"); } else if (marks >= 75) { console.log("Grade: A"); // Executes here } else { console.log("Grade: F"); }

4. Switch Statement

Used to select one of many code blocks to be executed. Note: break terminates the switch block, and default specifies the code to run if there is no case match.

let dayNumber = 2; let dayName; switch (dayNumber) { case 1: dayName = "Monday"; break; case 2: dayName = "Tuesday"; break; // Matches here default: dayName = "Invalid Day"; }

6. Loops

Loops are used to run the same block of code over and over again, as long as a specified condition is met.

1. for Loop

Loops through a block of code a specified number of times.

Syntax: for (initialization; condition; increment/decrement)

for (let i = 1; i <= 5; i++) { console.log("Number: " + i); }

2. while Loop

Loops through a block of code as long as a specified condition is true.

let count = 1; while (count <= 3) { console.log("Count is: " + count); count++; // Essential to avoid infinite loop }

3. do...while Loop

This loop will execute the code block once before checking if the condition is true, then it will repeat the loop as long as the condition is true.

let x = 10; do { console.log("Runs at least once!"); x++; } while (x < 5);

4. for...of Loop

Loops through the values of an iterable object (like an Array or String).

let fruits = ["Apple", "Banana", "Mango"]; for (let fruit of fruits) { console.log(fruit); }

5. for...in Loop

Loops through the properties (Keys) of an Object.

let car = { brand: "Tata", model: "Nexon", year: 2024 }; for (let key in car) { console.log(key + ": " + car[key]); }

7. Functions

A function is a reusable block of code designed to perform a specific task. You define it once and can call (invoke) it multiple times. Functions help avoid code repetition.

1. Function Declaration

The traditional way to define a function using the function keyword. It gets hoisted (can be called before it is defined in the code).

function sayHello() { console.log("Hello, Welcome to JavaScript!"); } sayHello(); // Calling the function

2. Function with Parameters

Parameters act as placeholders for values that you pass into the function when calling it.

function greetUser(name) { console.log("Hello, " + name + "! How are you?"); } greetUser("Noor"); // Outputs: Hello, Noor! How are you?

3. Function with Multiple Parameters

You can pass multiple values separated by commas.

function addNumbers(num1, num2) { console.log("Sum is: " + (num1 + num2)); } addNumbers(10, 20); // Outputs: Sum is: 30

4. Function with Return Value

The return statement sends a value back to where the function was called. The function stops executing when it hits return.

function multiply(a, b) { return a * b; // Returns the result } let result = multiply(5, 4); console.log("Multiplication Result: " + result); // Outputs: 20

5. Function Expression

Storing an anonymous function inside a variable. It is NOT hoisted — you must define it before calling.

const sayBye = function() { console.log("Goodbye! See you later."); } sayBye(); // Calling via variable name

6. Arrow Function (ES6)

A shorter, modern syntax for writing functions. It does not have its own this context. Note: If there is only one parameter, parentheses are optional. If there is only one return statement, curly braces and return keyword are optional.

// Full syntax const square = (num) => { return num * num; }; console.log(square(6)); // Outputs: 36 // Short syntax (implicit return) const doubleIt = (n) => n * 2; console.log(doubleIt(7)); // Outputs: 14

8. Arrays

An array is a single variable used to store multiple values in an ordered list. Arrays are zero-indexed, meaning the first element is at position 0. Arrays can hold mixed data types.

1. Creating an Array

Using square brackets [] is the most common way (Array Literal).

let colors = ["Red", "Green", "Blue"]; let mixed = ["Apple", 25, true, null]; // Mixed data types allowed

2. Accessing Elements by Index

Access elements using their index number inside square brackets.

let fruits = ["Mango", "Banana", "Grapes"]; console.log(fruits[0]); // Outputs: "Mango" (First element) console.log(fruits[2]); // Outputs: "Grapes" (Third element)

3. Modifying Elements

You can change an element's value by assigning a new value to a specific index.

let cars = ["Honda", "Toyota", "Ford"]; cars[1] = "BMW"; // Changing "Toyota" to "BMW" console.log(cars); // Outputs: ["Honda", "BMW", "Ford"]

4. Array Length Property

The .length property returns the total number of elements in the array.

let numbers = [10, 20, 30, 40, 50]; console.log(numbers.length); // Outputs: 5

5. Adding Elements

push() adds to the end. unshift() adds to the beginning.

let animals = ["Dog", "Cat"]; animals.push("Lion"); // Adds to end -> ["Dog", "Cat", "Lion"] animals.unshift("Bird"); // Adds to start -> ["Bird", "Dog", "Cat", "Lion"] console.log(animals);

6. Removing Elements

pop() removes from the end. shift() removes from the beginning.

let items = ["Pen", "Book", "Eraser", "Scale"]; items.pop(); // Removes "Scale" from end items.shift(); // Removes "Pen" from start console.log(items); // Outputs: ["Book", "Eraser"]

9. Objects

An object is a collection of related data stored as key-value pairs. The key (also called property name) is always a string, and the value can be any data type (string, number, array, function, or even another object). Objects are used to represent real-world entities.

1. Creating an Object (Object Literal)

The most common way — using curly braces {}.

let student = { name: "Noor", age: 22, course: "Web Development", isGraduated: false };

2. Accessing Properties (Dot Notation)

Use a dot . followed by the property name. Most common and easiest way.

let car = { brand: "Tata", model: "Nexon" }; console.log(car.brand); // Outputs: "Tata" console.log(car.model); // Outputs: "Nexon"

3. Accessing Properties (Bracket Notation)

Use square brackets with the property name as a string. Useful when property names have spaces or are stored in a variable.

let user = { firstName: "Rahul", "last name": "Sharma" }; console.log(user["firstName"]); // Outputs: "Rahul" console.log(user["last name"]); // Outputs: "Sharma" (Cannot use dot notation here)

4. Modifying Properties

Assign a new value to an existing key using dot or bracket notation.

let phone = { brand: "Samsung", price: 25000 }; phone.price = 22000; // Updated price console.log(phone.price); // Outputs: 22000

5. Adding New Properties

Simply assign a value to a new key that doesn't exist yet.

let laptop = { brand: "Dell", ram: "8GB" }; laptop.storage = "512GB"; // Adding new property laptop.color = "Black"; // Adding another new property console.log(laptop); // Outputs: { brand: "Dell", ram: "8GB", storage: "512GB", color: "Black" }

6. Deleting Properties

Use the delete keyword to remove a property entirely from the object.

let employee = { name: "Priya", dept: "HR", salary: 40000 }; delete employee.salary; // Removes the salary property console.log(employee); // Outputs: { name: "Priya", dept: "HR" }

7. Object with Method (Function inside Object)

A method is a function stored as a property inside an object. It allows the object to perform actions using its own data. Use this keyword to refer to the current object's properties inside the method.

let person = { name: "Amit", age: 30, introduce: function() { console.log("Hi, I am " + this.name + " and I am " + this.age + " years old."); } }; person.introduce(); // Outputs: Hi, I am Amit and I am 30 years old.

10. Array Methods

Array methods are built-in functions that you can call on arrays to perform common operations like searching, transforming, filtering, and sorting — without writing manual loops.

1. indexOf()

Returns the first index at which a given element is found. Returns -1 if the element is not present.

let fruits = ["Apple", "Mango", "Banana", "Mango"]; console.log(fruits.indexOf("Mango")); // Outputs: 1 (first occurrence) console.log(fruits.indexOf("Grapes")); // Outputs: -1 (not found)

2. includes()

Returns true if the array contains the specified element, otherwise false.

let colors = ["Red", "Green", "Blue"]; console.log(colors.includes("Green")); // Outputs: true console.log(colors.includes("Yellow")); // Outputs: false

3. join()

Joins all elements of an array into a single string, separated by a specified separator (default is comma).

let words = ["Hello", "World", "JavaScript"]; console.log(words.join()); // Outputs: "Hello,World,JavaScript" console.log(words.join(" - ")); // Outputs: "Hello - World - JavaScript" console.log(words.join(" ")); // Outputs: "Hello World JavaScript"

4. slice()

Returns a shallow copy of a portion of the array into a new array. Note: It does NOT modify the original array. Parameters are (start, end) — end index is excluded.

let nums = [10, 20, 30, 40, 50]; let sliced = nums.slice(1, 4); // Index 1 to 3 (4 is excluded) console.log(sliced); // Outputs: [20, 30, 40] console.log(nums); // Original unchanged: [10, 20, 30, 40, 50]

5. splice()

Used to add, remove, or replace elements in an array. Note: It MODIFIES the original array. Parameters: splice(startIndex, deleteCount, item1ToAdd, item2ToAdd, ...).

let arr = ["A", "B", "C", "D", "E"]; // Remove 2 elements starting from index 1 let removed = arr.splice(1, 2); console.log(removed); // Outputs: ["B", "C"] (removed items) console.log(arr); // Outputs: ["A", "D", "E"] (original modified)

6. concat()

Merges two or more arrays and returns a new array. Does NOT change the original arrays.

let arr1 = [1, 2, 3]; let arr2 = [4, 5, 6]; let merged = arr1.concat(arr2); console.log(merged); // Outputs: [1, 2, 3, 4, 5, 6] console.log(arr1); // Outputs: [1, 2, 3] (unchanged)

7. reverse()

Reverses the order of elements in the array. Warning: It MODIFIES the original array.

let letters = ["A", "B", "C", "D"]; letters.reverse(); console.log(letters); // Outputs: ["D", "C", "B", "A"]

8. sort()

Sorts the elements of an array in-place (modifies original). By default, it sorts elements as strings alphabetically, which can cause issues with numbers.

// String sorting (default) let names = ["Zara", "Noor", "Amit"]; names.sort(); console.log(names); // Outputs: ["Amit", "Noor", "Zara"] // Number sorting (needs compare function) let numArr = [40, 100, 1, 5, 25]; numArr.sort(function(a, b) { return a - b; }); // Ascending console.log(numArr); // Outputs: [1, 5, 25, 40, 100]

9. push() & pop()

push() adds one or more elements to the end and returns the new length. pop() removes the last element and returns it.

let stack = [10, 20]; let newLength = stack.push(30, 40); // Adds to end console.log(newLength); // Outputs: 4 let lastItem = stack.pop(); // Removes from end console.log(lastItem); // Outputs: 40 console.log(stack); // Outputs: [10, 20, 30]

10. shift() & unshift()

shift() removes the first element and returns it. unshift() adds elements to the beginning and returns the new length.

let queue = ["B", "C", "D"]; queue.unshift("A"); // Adds "A" to start -> ["A", "B", "C", "D"] let first = queue.shift(); // Removes "A" from start console.log(first); // Outputs: "A" console.log(queue); // Outputs: ["B", "C", "D"]

11. map()

Creates a new array by applying a function to every element of the original array. Does NOT modify the original. Commonly used for transforming data.

let prices = [100, 200, 300]; let discounted = prices.map(function(price) { return price - 20; // Apply 20 rupees discount }); console.log(discounted); // Outputs: [80, 180, 280] console.log(prices); // Outputs: [100, 200, 300] (unchanged)

12. filter()

Creates a new array containing only those elements that pass a test (condition returns true). Does NOT modify the original.

let ages = [12, 18, 25, 8, 30, 16]; let adults = ages.filter(function(age) { return age >= 18; // Keep only 18 or above }); console.log(adults); // Outputs: [18, 25, 30] console.log(ages); // Outputs: [12, 18, 25, 8, 30, 16] (unchanged)

13. reduce()

Reduces the entire array to a single value (like a sum, product, or merged string). It takes a callback with an accumulator and a current value.

let numbers = [1, 2, 3, 4, 5]; let total = numbers.reduce(function(accumulator, currentValue) { return accumulator + currentValue; }, 0); // 0 is the initial value of accumulator console.log(total); // Outputs: 15 (1+2+3+4+5)

14. find()

Returns the first element that satisfies the provided condition. Returns undefined if no element matches.

let users = [ { id: 1, name: "Noor" }, { id: 2, name: "Rahul" }, { id: 3, name: "Priya" } ]; let found = users.find(function(user) { return user.id === 2; }); console.log(found); // Outputs: { id: 2, name: "Rahul" }

15. findIndex()

Returns the index of the first element that satisfies the condition. Returns -1 if not found.

let scores = [45, 80, 55, 90, 60]; let index = scores.findIndex(function(score) { return score > 85; }); console.log(index); // Outputs: 3 (because 90 is at index 3)

16. forEach()

Executes a provided function once for each array element. It does NOT return a new array (returns undefined). Used mainly for performing side effects like logging.

let cities = ["Delhi", "Mumbai", "Chennai"]; cities.forEach(function(city, index) { console.log(index + ": " + city); }); // Outputs: // 0: Delhi // 1: Mumbai // 2: Chennai

17. every()

Returns true if ALL elements pass the test condition. Returns false if even one element fails.

let marks = [80, 85, 90, 78, 88]; let allPassed = marks.every(function(mark) { return mark >= 75; // Check if everyone scored 75+ }); console.log(allPassed); // Outputs: true

18. some()

Returns true if AT LEAST ONE element passes the test condition. Opposite of every().

let ages = [12, 15, 22, 10]; let hasAdult = ages.some(function(age) { return age >= 18; }); console.log(hasAdult); // Outputs: true (because 22 is present)

19. flat()

Flattens a nested array (array inside array) into a single flat array. Takes an optional depth argument (default is 1).

let nested = [1, [2, 3], [4, [5, 6]]]; console.log(nested.flat()); // Outputs: [1, 2, 3, 4, [5, 6]] (depth 1) console.log(nested.flat(Infinity)); // Outputs: [1, 2, 3, 4, 5, 6] (fully flat)

20. Array.isArray()

A static method used to check if a given value is an array. Returns true or false.

let arr = [1, 2, 3]; let obj = { name: "Noor" }; console.log(Array.isArray(arr)); // Outputs: true console.log(Array.isArray(obj)); // Outputs: false

11. Object Methods

Object methods are built-in functions that help you work with objects — like getting keys, values, converting objects to arrays, and merging objects.

1. Object.keys()

Returns an array of all keys (property names) of the object.

let student = { name: "Noor", age: 22, city: "Delhi" }; let keys = Object.keys(student); console.log(keys); // Outputs: ["name", "age", "city"]

2. Object.values()

Returns an array of all values of the object's properties.

let car = { brand: "Tata", model: "Nexon", year: 2024 }; let vals = Object.values(car); console.log(vals); // Outputs: ["Tata", "Nexon", 2024]

3. Object.entries()

Returns an array of arrays, where each inner array is a [key, value] pair. Useful when you want to loop through both keys and values together.

let product = { name: "Laptop", price: 50000, inStock: true }; let entries = Object.entries(product); console.log(entries); // Outputs: [ ["name", "Laptop"], ["price", 50000], ["inStock", true] ]

4. Object.hasOwnProperty()

Checks if the object has a specific property as its own (not inherited). Returns true or false.

let user = { name: "Amit", email: "amit@mail.com" }; console.log(user.hasOwnProperty("name")); // Outputs: true console.log(user.hasOwnProperty("phone")); // Outputs: false

5. Object.assign()

Copies properties from one or more source objects into a target object. Modifies the target object and also returns it.

let target = { a: 1, b: 2 }; let source = { b: 3, c: 4 }; // 'b' will overwrite target's 'b' let result = Object.assign(target, source); console.log(result); // Outputs: { a: 1, b: 3, c: 4 } console.log(target); // Outputs: { a: 1, b: 3, c: 4 } (target is modified)

6. Object.freeze()

Makes an object completely immutable — you cannot add, remove, or modify any properties. Returns the frozen object.

let config = { theme: "dark", fontSize: 14 }; Object.freeze(config); config.theme = "light"; // Silently fails (no error in non-strict mode) config.newProp = "test"; // Silently fails console.log(config); // Outputs: { theme: "dark", fontSize: 14 } (unchanged)

7. Object.seal()

Prevents adding new properties or removing existing ones, but allows modifying existing property values.

let data = { name: "Rahul", score: 85 }; Object.seal(data); data.score = 90; // Allowed: modifying existing property data.city = "Mumbai"; // Silently fails: cannot add new property delete data.name; // Silently fails: cannot delete console.log(data); // Outputs: { name: "Rahul", score: 90 }

8. Object.keys().length

Objects don't have a .length property directly. To find how many properties an object has, use Object.keys(obj).length.

let employee = { id: 1, name: "Priya", dept: "HR", salary: 40000 }; let propCount = Object.keys(employee).length; console.log(propCount); // Outputs: 4

12. String Methods

String methods are built-in functions that help you manipulate, search, extract, and transform text. Important: Strings are immutable in JavaScript — no method changes the original string; they always return a new string.

1. length (Property)

Not a method, but a property. Returns the total number of characters in the string (spaces are also counted).

let str = "Hello World"; console.log(str.length); // Outputs: 11

2. toUpperCase()

Converts the entire string to uppercase letters.

let name = "noor"; console.log(name.toUpperCase()); // Outputs: "NOOR"

3. toLowerCase()

Converts the entire string to lowercase letters.

let city = "DELHI"; console.log(city.toLowerCase()); // Outputs: "delhi"

4. trim()

Removes whitespace (spaces, tabs, newlines) from both ends of the string. Does NOT remove spaces from the middle.

let messy = " Hello World "; console.log(messy.trim()); // Outputs: "Hello World"

5. trimStart()

Removes whitespace only from the beginning (left side) of the string.

let text = " Hello World"; console.log(text.trimStart()); // Outputs: "Hello World"

6. trimEnd()

Removes whitespace only from the end (right side) of the string.

let text = "Hello World "; console.log(text.trimEnd()); // Outputs: "Hello World"

7. includes()

Checks if a substring exists inside the string. Returns true or false. It is case-sensitive.

let sentence = "I love JavaScript"; console.log(sentence.includes("love")); // Outputs: true console.log(sentence.includes("Love")); // Outputs: false (case-sensitive)

8. indexOf()

Returns the first index (position) where the specified substring is found. Returns -1 if not found.

let msg = "Hello World Hello"; console.log(msg.indexOf("Hello")); // Outputs: 0 (first occurrence) console.log(msg.indexOf("World")); // Outputs: 6 console.log(msg.indexOf("Bye")); // Outputs: -1 (not found)

9. lastIndexOf()

Same as indexOf(), but searches from the end and returns the last occurrence's index.

let msg = "Hello World Hello"; console.log(msg.lastIndexOf("Hello")); // Outputs: 12 (second "Hello")

10. startsWith()

Checks if the string starts with the specified substring. Returns true or false.

let url = "https://google.com"; console.log(url.startsWith("https")); // Outputs: true console.log(url.startsWith("http")); // Outputs: false

11. endsWith()

Checks if the string ends with the specified substring. Returns true or false.

let filename = "photo.jpg"; console.log(filename.endsWith(".jpg")); // Outputs: true console.log(filename.endsWith(".png")); // Outputs: false

12. charAt()

Returns the character at a specific index. If the index is out of range, it returns an empty string.

let word = "JavaScript"; console.log(word.charAt(0)); // Outputs: "J" console.log(word.charAt(4)); // Outputs: "S"

13. charCodeAt()

Returns the Unicode (ASCII) number of the character at a given index.

let letter = "A"; console.log(letter.charCodeAt(0)); // Outputs: 65 console.log("a".charCodeAt(0)); // Outputs: 97

14. slice(start, end)

Extracts a portion of the string and returns it as a new string. The end index is excluded. Negative values count from the end.

let str = "Hello World"; console.log(str.slice(0, 5)); // Outputs: "Hello" (index 0 to 4) console.log(str.slice(6)); // Outputs: "World" (from index 6 to end) console.log(str.slice(-5)); // Outputs: "World" (last 5 characters)

15. substring(start, end)

Same as slice() but does NOT accept negative indices. If you pass a negative number, it treats it as 0.

let str = "Hello World"; console.log(str.substring(0, 5)); // Outputs: "Hello" console.log(str.substring(6, 11)); // Outputs: "World" console.log(str.substring(-3)); // Outputs: "Hello World" (negative treated as 0)

16. replace()

Replaces the first occurrence of a substring with another string. It is case-sensitive and does NOT modify the original string.

let text = "I love cats. Cats are cute."; console.log(text.replace("cats", "dogs")); // Outputs: "I love dogs. Cats are cute." (only first "cats" replaced)

17. replaceAll()

Replaces ALL occurrences of a substring. (ES2021 feature — works in modern browsers).

let text = "I love cats. Cats are cute."; console.log(text.replaceAll("cats", "dogs").replaceAll("Cats", "Dogs")); // Outputs: "I love dogs. Dogs are cute."

18. split()

Splits a string into an array of substrings based on a specified separator. Very useful for breaking strings into words or parts.

let csv = "Red,Green,Blue,Yellow"; console.log(csv.split(",")); // Outputs: ["Red", "Green", "Blue", "Yellow"] let sentence = "Hello World JavaScript"; console.log(sentence.split(" ")); // Outputs: ["Hello", "World", "JavaScript"] let word = "JavaScript"; console.log(word.split("")); // Outputs: ["J","a","v","a","S","c","r","i","p","t"]

19. repeat()

Returns a new string with the original string repeated a specified number of times.

let dash = "-"; console.log(dash.repeat(10)); // Outputs: "----------" let ha = "Ha"; console.log(ha.repeat(3)); // Outputs: "HaHaHa"

20. concat()

Joins two or more strings together and returns a new string. Same as using the + operator.

let first = "Hello"; let second = "World"; console.log(first.concat(" ", second)); // Outputs: "Hello World" console.log(first.concat(" ", second, "!")); // Outputs: "Hello World!"

21. padStart()

Pads the string from the start with a specified character until it reaches the target length. Useful for formatting numbers.

let num = "5"; console.log(num.padStart(3, "0")); // Outputs: "005" let pin = "123"; console.log(pin.padStart(6, "*")); // Outputs: "***123"

22. padEnd()

Pads the string from the end with a specified character until it reaches the target length.

let name = "Noor"; console.log(name.padEnd(10, ".")); // Outputs: "Noor......" let price = "99"; console.log(price.padEnd(6, " ")); // Outputs: "99 "

23. match()

Searches the string using a regular expression (RegExp) and returns an array of matches, or null if no match is found.

let text = "Call me at 9876543210 or 9123456789"; let numbers = text.match(/\d{10}/g); // Find all 10-digit numbers console.log(numbers); // Outputs: ["9876543210", "9123456789"]

13. DOM Manipulation

DOM (Document Object Model) is a tree-like structure created by the browser when an HTML page loads. JavaScript uses the DOM to select, read, modify, add, and remove HTML elements dynamically — without reloading the page. Think of the DOM as JavaScript's "remote control" for the webpage.

1. Selecting Elements

The first step in DOM manipulation is finding the element you want to work with. There are 4 main methods:

// 1. getElementById — Returns a SINGLE element by its ID let heading = document.getElementById("main-title"); // 2. getElementsByClassName — Returns a LIVE HTMLCollection of elements let cards = document.getElementsByClassName("card"); // 3. getElementsByTagName — Returns a LIVE HTMLCollection by tag name let paragraphs = document.getElementsByTagName("p"); // 4. querySelector — Returns the FIRST element matching a CSS selector (most versatile) let firstBtn = document.querySelector(".btn-primary"); let navLink = document.querySelector("nav a"); // 5. querySelectorAll — Returns a STATIC NodeList of ALL matching elements let allItems = document.querySelectorAll(".list-item");

2. Reading & Changing Content

Once you select an element, you can read or change its content using these properties:

let box = document.getElementById("box"); // innerHTML — Gets/sets the HTML inside the element (parses HTML tags) box.innerHTML = "<strong>Hello</strong> World"; // Renders "Hello" in bold // textContent — Gets/sets only the plain text (ignores HTML tags, safer) box.textContent = "<strong>Hello</strong> World"; // Shows tags as plain text // innerText — Similar to textContent but respects CSS visibility (hidden text is excluded) console.log(box.innerText);

3. Changing Attributes

You can get, set, or remove HTML attributes (like src, href, class, disabled, etc.) using these methods:

let img = document.querySelector("img"); let link = document.querySelector("a"); // getAttribute — Read an attribute's value console.log(img.getAttribute("src")); // Outputs: "photo.jpg" // setAttribute — Set or change an attribute's value img.setAttribute("src", "new-photo.jpg"); link.setAttribute("href", "https://google.com"); // removeAttribute — Remove an attribute completely img.removeAttribute("alt"); // Direct property access (shorter, works for standard attributes) console.log(img.src); img.src = "another-photo.jpg"; link.href = "https://yahoo.com";

4. Changing Styles

Use the style property to change inline CSS. Property names use camelCase (not kebab-case).

let card = document.querySelector(".card"); // Single style change card.style.backgroundColor = "#f7df1e"; card.style.color = "#333"; card.style.padding = "20px"; card.style.borderRadius = "8px"; card.style.display = "none"; // Hide element // CSS classes approach (recommended for multiple styles) card.classList.add("highlight"); // Add a class card.classList.remove("highlight"); // Remove a class card.classList.toggle("active"); // Add if missing, remove if present card.classList.contains("active"); // Returns true/false card.classList.replace("old", "new"); // Replace one class with another

5. Creating & Adding Elements

You can create brand new elements and insert them into the DOM:

// Step 1: Create the element let newParagraph = document.createElement("p"); // Step 2: Add content and attributes newParagraph.textContent = "This is a newly created paragraph."; newParagraph.id = "new-para"; newParagraph.classList.add("text-muted"); // Step 3: Insert into the DOM let container = document.getElementById("container"); container.appendChild(newParagraph); // Adds as the LAST child // Other insertion methods (relative positioning) let referenceNode = document.querySelector(".existing-item"); container.insertBefore(newParagraph, referenceNode); // Insert BEFORE another element

6. Removing Elements

To remove an element from the DOM, you need to select it AND its parent:

// Method 1: Using removeChild (traditional) let parent = document.getElementById("list"); let child = document.getElementById("item-3"); parent.removeChild(child); // Removes child from parent // Method 2: Using remove() (modern, cleaner — no need for parent reference) let oldElement = document.querySelector(".outdated"); oldElement.remove(); // Directly removes the element

7. Traversing the DOM

Moving between related elements (parent, children, siblings) in the DOM tree:

let item = document.querySelector(".item"); // Parent let parent = item.parentElement; // or item.parentNode // Children let children = item.children; // HTMLCollection (only element nodes) let firstChild = item.firstElementChild; // First child element let lastChild = item.lastElementChild; // Last child element // Siblings let nextSibling = item.nextElementSibling; // Next element sibling let prevSibling = item.previousElementSibling; // Previous element sibling Note: Use elementChildren/ElementSibling versions to skip text nodes (whitespace).

8. Working with Classes (classList)

classList is the recommended way to manipulate CSS classes. It is more efficient than changing styles one by one.

let btn = document.querySelector("button"); // Add one or more classes btn.classList.add("btn-primary", "rounded", "shadow"); // Remove specific classes btn.classList.remove("rounded"); // Toggle — adds if not present, removes if present (very useful for dark mode, menus, etc.) btn.classList.toggle("active"); // Check if a class exists if (btn.classList.contains("btn-primary")) { console.log("Primary button found!"); } // Replace a class with another btn.classList.replace("btn-primary", "btn-danger");

9. Data Attributes (dataset)

Data attributes (data-*) let you store custom information in HTML elements. Access them via dataset in JS.

<!-- HTML --> <div id="user" data-userid="101" data-role="admin" data-full-name="Noor Ahmed"></div> // JavaScript let userDiv = document.getElementById("user"); console.log(userDiv.dataset.userid); // Outputs: "101" console.log(userDiv.dataset.role); // Outputs: "admin" console.log(userDiv.dataset.fullName); // Outputs: "Noor Ahmed" (kebab-case becomes camelCase) // Set new data attribute userDiv.dataset.status = "online"; // HTML becomes: data-status="online"
